# expense_tracker
Source code for the Expense tracker is in the Python_Prototype folder and all the different versions aswell. 

Executable was compiled with pyinstaller on Pop OS and on similar linux operation systems it should be opened by opening a terminal in the directly of the executable and typing ./Expense_Tracker.
This executable will create multiple files and most importantly Theirs a .KEY file that should only be deleted to reset because of a forgotten passkey. Otherwise you can reset App data in the other options menu.
